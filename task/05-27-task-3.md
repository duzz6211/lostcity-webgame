목표 확장 카드 목록

1. 선착순: 아무 탐험지에나 카드 5장 먼저 놓기, 10점 추가

2. 선착순: 카드가 1장 이상 놓인 탐험지 3개 이상, 10점 추가

3. 선착순: 하나의 탐험에 이어지는 숫자 3개 (연속 숫자), 10점 추가

4. 게임 종료 시: 카드가 1장 이상 놓인 탐험이 더 많은 플레이어 (동점일 시, 무효), 10점 추가

5. 게임 종료: 연구 카드가 없는 탐험의 최고 점수가 더 높은 플레이어, 10점 추가

6. 게임 종료: 손에 든 탐험 카드의 숫자 총합이 더 낮은 플레이어, 10점 추가

7. 선착순: 노란색 탐험지에 3장 먼저 놓은 플레이어, 15점 추가

8. 선착순: 파란색과 파란색이 아닌 아무 탐험지에 숫자 총합 10점을 먼저 넘긴 플레이어, 10점 추가

9. 게임 종료: 손에 든 악수카드가 더 많은 플레이어, 10점 추가

10. 게임 종료: 손에 든 초록색 카드가 더 많은 플레이어, 10점 추가

11. 선착순: 모든 탐험지에 하나 이상 둔 플레이어, 20점 추가

선착순: 먼저 달성한 플레이어가 얻어가는 점수
게임 종료: 한 라운드가 종료 된 시점에서 판단하는 점수
